API Documentation

Integrate SkillPort into your applications with our RESTful API.

Authentication

All API requests require authentication using API keys.

Authorization: Bearer YOUR_API_KEY

Base URL

All API endpoints are relative to:

https://api.skillport.com/v1

Rate Limits

Free tier: 100 requests/hour
Pro tier: 1000 requests/hour
Enterprise: Custom limits

SDKs and Libraries

JavaScript/Node.js

Official SDK for JavaScript and Node.js applications.

npm install skillport-sdk

Python

Python client library for SkillPort API.

pip install skillport-python

Code Examples

Get User Profile

GET /users/profile
Authorization: Bearer YOUR_API_KEY

Response:
{
  "id": "user_123",
  "username": "john_doe",
  "email": "john@example.com",
  "progress": {
    "problems_solved": 150,
    "contests_won": 5
  }
}
